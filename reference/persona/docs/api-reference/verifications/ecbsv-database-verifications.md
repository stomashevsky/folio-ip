# eCBSV Database Verifications

An eCBSV Verification connects to the Social Security Administration’s electronic Consent Based Social Security Number Verification (eCBSV) Service. This verification answers the question “Do the SSN, Name, and Birthdate match against SSA’s data records?”. This is especially useful when combined with a Database verifications to accept thin-file applicants or detect synthetic identities.
