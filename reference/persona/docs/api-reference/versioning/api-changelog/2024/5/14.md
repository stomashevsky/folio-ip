### Document Verifications

-   ğŸƒ **Add `document-type` Document Verifications**: Document Verifications now include a `document-type` attribute. `document-type` refers to the classification of the submitted document, such as `bank_statement` or `credit_card_statement`.

### Inquiries

-   ğŸƒ **Asynchronous redaction of related resources**: Previously, the [Inquiry redaction API](../../../../inquiries/redact-an-inquiry.md) would redact the Inquiry as well as any related resources (Accounts, Verifications, etc.) synchronously. Now it will redact the Inquiry synchronously and redact any related resources asynchronously. The `inquiry.redacted` event will only be emitted after all objects are redacted. This should improve performance.
