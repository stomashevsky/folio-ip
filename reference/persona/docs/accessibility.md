# Accessibility

Persona is committed to providing a flow that is accessible to all users. We are continually improving the user experience and ensuing that we adhere to public accessibility standards such as [WCAG](https://www.w3.org/TR/WCAG22/) regarding color, text sizing, keyboard navigation, and screen reader support.

## Screen readers

Persona aims to support the following screen readers:

-   All built-in screen readers for our supported browsers and devices (e.g. Apple VoiceOver, Android TalkBack, Microsoft Narrator)
-   [NVDA](https://www.nvaccess.org/)
-   [JAWS](https://support.freedomscientific.com/JAWSHQ/JAWSHeadquarters01)

## Reporting issues

We are continually improving the accessibility of our flow. If you would like to report an issue, please [contact us](https://app.withpersona.com/dashboard/contact-us).
