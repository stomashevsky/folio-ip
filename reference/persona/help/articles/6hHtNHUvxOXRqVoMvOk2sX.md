# Cases Field Editor

# Overview[](#overview)

To ensure that your organization has all the relevant data you need for Cases, add or edit fields via Cases Field Editor.

This unlocks a more seamless experience that enables you to:

-   Create, edit, deactivate, or delete fields
-   Set default field values and display them in the table view
-   Search by name and keywords and filter by a fieldâ€™s data type

![Cases Field Editor](../images/Cases_Field_Editor_16ea1ea5bf4b.png)

## **Where can you find the Cases Field Editor?**[](#where-can-you-find-the-cases-field-editor)

1.  In the Dashboard, click **Cases** > **Templates** in the navigation bar.
2.  After [adding a Case template or selecting an existing one](./5WFMyVPjzgXQNljqg2xf4h.md), click Settings then Fields to open the Field Editor.

## **How to add or edit Case fields?**[](#how-to-add-or-edit-case-fields)

1.  Navigate to the Cases Field Editor.
2.  Click into an existing field or click Add Field
3.  Update Field Details:
    1.  **Field label:** the name of the custom field (displayed within the dashboard)
    2.  **Field key:** a unique identifier for the field (also used when retrieving or updating the object via API)
    3.  **Field type:** describes the type of field value allowed, such as an array, date, integer, string, etc.
