# Database Verification (Turkey)

_The ability to use the pre-built Database Verification (TR) solution is available on select plans. Customers can contact your Customer Success Manager or [Customer Support](https://app.withpersona.com/dashboard/contact-us) for more information._

## Database Verification (TR) Features[](#database-verification-tr-features)

Persona's Database Verification product offers the ability to verify attributes of an individual against authoritative Turkish databases. This helps your business gain an understanding of users associated with Turkey. Using the out-of-the-box Turkish Database Verification, you can verify attributes like name, date of birth, address, and certain national identification numbers against known matches in authoritative databases.

Learn more about [Database Verification](./7dcdaIsMttmPUZ2ZelRbZD.md).
