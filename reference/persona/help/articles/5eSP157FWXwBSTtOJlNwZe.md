# How do I tell if I'm using a Dynamic Flow Inquiry Template?

To tell if a Template is a Dynamic Flow Template, look at its ID:

-   If the Template ID starts with `itmpl_`, it's a Dynamic Flow Template.
-   If the Template ID starts with `tmpl_`, it's the older-style Template.

The Inquiry Templates view of the Persona Dashboard displays the ID of each Template. To get to this page, select **Inquiries > Templates** from the navigation bar.
