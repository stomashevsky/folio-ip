# Why should you use Accounts to store all of a single individual's PII and information?

By consolidating multiple Inquiries for given user in a single Account, you can search for all information collected over time as part of a users' identity verification and quickly respond to data subject access requests or data deletion requests. This also includes all **supplemental documents** collected over time. To learn more, check out our [blog post on this topic](../../landing/blog/pii-accounts.md).
