# What do the various Database Verification statuses mean?

## Overview[](#overview)

The status of a Database Verification indicates whether it's just starting, being processed, or has a result.

## Statuses[](#statuses)

-   **Initiated**: The starting status.
-   **Submitted**: The required fields have been submitted to the Database Verification, and are saved in the Persona system. Results are processing.
-   **Passed**: The Database Verification passed, according to the configured Verification checks.
-   **Failed**: The Database Verification failed, according to the configured Verification checks.
-   **Canceled**: The Database Verification was submitted, but did not complete. No result was determined.
