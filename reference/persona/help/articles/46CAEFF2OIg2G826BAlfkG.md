# Transactions Field Editor

# Overview[](#overview)

To ensure that your organization has all the relevant data you need for Transactions, add or edit fields via Transactions Field Editor.

This unlocks a more seamless experience that enables you to:

-   Create, edit, deactivate, or delete fields
-   Create a relation
-   Set default field values and display them in the table view
-   Search by name and keywords and filter by a fieldâ€™s data type

![Transactions Field Editor](../images/Transactions_Field_Editor_eb1814f3f480.png)

## Where can you find the Transactions Field Editor?[](#where-can-you-find-the-transactions-field-editor)

1.  In the Dashboard, click **Transactions > Types** in the navigation bar.
2.  After [adding a Transaction Type or selecting an existing one](./2QtlaeVrfaokLJnLZ8cRNL.md), the Field Editor will appear.

## How to add or edit Transactions fields?[](#how-to-add-or-edit-transactions-fields)

1.  Navigate to the Transactions Field Editor.
2.  Click into an existing field or click Add Field
3.  Update Field Details:
    1.  **Field label:** the name of the custom field (displayed within the dashboard)
    2.  **Field key:** a unique identifier for the field (also used when retrieving or updating the object via API)
    3.  **Field type:** describes the type of field value allowed, such as an array, date, integer, string, etc.
